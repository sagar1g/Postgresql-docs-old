1. Single-Column Index:
This is the most common type of index, where an index is created on a single column of a table.
command:-
CREATE INDEX index_name ON table_name (column_name);

2.Multi-Column Index:
You can create an index on multiple columns, which can be useful for queries involving multiple columns.
command:-
CREATE INDEX index_name ON table_name (column1_name, column2_name);

3. Partial Index:
A partial index is an index that includes only a subset of the rows in a table based on a specified condition

CREATE INDEX index_name ON table_name (column_name) WHERE condition;

4.Unique Index:
A unique index ensures that the indexed column(s) contain unique values.
command:-
CREATE UNIQUE INDEX index_name ON table_name (column_name);

5.B-tree Index:
B-tree indexes are the default index type in PostgreSQL, suitable for range queries.
command:-
CREATE INDEX index_name ON table_name USING btree (column_name);
